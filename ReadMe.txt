Navigation&Lighting

==================================================
功能

1.建立地形，並加入Nav mesh，調整參數使產生的路徑範為只有平地
2.加入鏡頭光斑和LightCookie
3.角色能自動從起點到終點
4.多個角色(至少5個)一起走到終點，彼此之間會互相閃躲
5.Camera跟隨其中一個角色
6.使用Trigger，當其中一個角色經過後觸發機關，產生會動的障礙物阻擋角色移動路徑
7.角色會自動避開障礙物
==================================================
操作

不用操作
